# Automação com Python para Iniciantes

Landing page oficial do eBook **Automação com Python para Iniciantes**.

## Sobre o eBook

Este eBook ensina como criar automações simples utilizando Python, mesmo para quem nunca programou.

Conteúdos abordados:

- O que é automação
- Por que Python é ideal para automação
- Instalação do Python
- Primeiro script automático
- Automação de mouse e teclado
- Automação de tarefas no computador
- Automação na internet
- Automação de planilhas
- Bots simples
- Ideias para ganhar dinheiro com automação

Bônus incluídos:

- Introdução ao Git
- Projetos práticos
- Robô de produtividade
- Dicas de mercado
